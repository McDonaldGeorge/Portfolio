Project Overview

Custom Chatbot is an end-to-end conversational AI project designed for a tailored domain and tone. The system combines a pre-trained transformer backbone with fine-tuned dialogue data, a lightweight API (Flask/FastAPI), and a web chat interface. The aim is to demonstrate practical conversational AI engineering: dataset preparation, model integration, prompt control, and deployment.

Python NLP Transformers Fine-Tuning Flask/FastAPI Vector Store (RAG) HTML/CSS/JS

Architecture

  • Frontend: Minimal HTML/CSS/JS chat interface with message history.
  • Backend API: Flask or FastAPI endpoint (POST /chat) that receives user input and returns model responses.
  • Model Layer: Pre-trained transformer; optional fine-tuning for tone/domain.
  • Memory: Session context; optional vector database for retrieval-augmented generation (RAG).
  • Deployment: Static frontend (GitHub Pages) + API on a serverless/cloud host.
